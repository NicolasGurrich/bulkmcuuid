# Bulk MC UUID Converter
Python File that converts a bulk amount of Minecraft Usernames to UUIDs

Enter Usernames into usernames.txt - one on each line - and run conv.py
UUIDs will be stored in uuids.txt and rejected usernames will be sent to not_found.txt
